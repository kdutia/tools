How to Use

  • Start Recording: Click the button or press Enter to begin capturing
  • Type rhythms: Press any keys to record. Space is tracked separately from all other keys
  • Stop Recording: Click the button again or press Escape
  • Export: Choose MIDI for DAWs, JSON for detailed data, or CSV for spreadsheets
  • MIDI Notes: Customize which notes each key type triggers in your DAW

Tips for Music Production

  • Use Space for kick drum or bass rhythm
  • Use other keys for hi-hats, snares, or melodic notes
  • The timing resolution is ~0.1ms, capturing subtle groove variations
  • Import the MIDI into your DAW and quantize if needed, or keep the human feel
